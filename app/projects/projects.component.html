Quiz Games

  • SentenceBuilder
    A Quiz I originally made as part of an MVC/AngularJS project together with Epoxie.
    As the name suggests it's a quiz game where you're supposed to build sentences, but I thought it would be a bit more interesting to use tongue twisters instead of "normal" sentences. I guess the code could use some "cleaning", but it'll stay on my to-do list for the time being.

  • PunctuationExercise
    Another Quiz I originally made as part of the same project together with Epoxie.
    As the name suggests it's a quiz game where you place punctuation marks at the correct places in a sentence, and this one also uses tongue twisters instead of "normal" sentences. Like SentenceBuilder the code for this one could also use some "cleaning", but it'll also stay on my to-do list for the time being.
